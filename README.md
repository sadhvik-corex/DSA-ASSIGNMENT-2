## 📖 Description

This repository contains C programs demonstrating applications of Stack and Queue using dynamic memory allocation (malloc).

All programs are implemented using linked list concepts instead of static arrays to provide flexibility and better memory utilization.

## 📂 Programs Included

1. Reverse String using Stack  
2. Balanced Parentheses  
3. Next Greater Element  
4. Printer Queue Simulation  
5. Circular Queue
